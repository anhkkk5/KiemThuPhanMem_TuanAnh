## 1. Mô tả bài toán
Chương trình phân tích điểm số học sinh:
- Đếm số học sinh đạt loại Giỏi (>= 8.0)
- Tính điểm trung bình của các điểm hợp lệ (0–10)
- Bỏ qua dữ liệu sai (<0 hoặc >10)

## 2. Công nghệ sử dụng
- Java
- JUnit 5
- GitHub Issues & Commit Convention
